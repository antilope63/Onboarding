## Couleurs — utilisez les variables prédéfinies

Règle simple: pas de codes hex/rgb/hsl en dur. Utilisez la palette existante.

### Comment utiliser

- bg-violet_fonce_1
- text-violet_fonce_1
- border-violet_fonce_1
- ...

### Où sont définies les couleurs

- `src/app/globals.css`

### Couleurs de marque

- noir
- violet_fonce_1
- violet
- bleu_fonce_1
- bleu_fonce_2

### Aperçu

- Page: `/colors`

### Démarrer

```bash
pnpm dev
```

App: `http://localhost:3000`
